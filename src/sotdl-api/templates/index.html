Here are the endpoints for creating a Shadow of the Demon Lord character. With any of the ancestries you can add a name to the end of the URL to roll a character with a permanent link.

  • GET /changeling
  • GET /changeling/{name}
  • GET /clockwork
  • GET /clockwork/{name}
  • GET /dwarf
  • GET /dwarf/{name}
  • GET /faun
  • GET /faun/{name}
  • GET /goblin
  • GET /goblin/{name}
  • GET /halfling
  • GET /halfling/{name}
  • GET /human
  • GET /human/{name}
  • GET /orc
  • GET /orc/{name}

Or, if you're feeling particularly adventurous, use the random endpoint!

  • GET /random/
